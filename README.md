# skool-2
# Installation guide
## 1. Clone the repository
```bash
git clone
```
## 2. Install dependencies
```bash
npm install
```
## 3. Start the server
```bash
npm run dev
```
## 4. Start the client
```bash 
npm start
```
## 5. Open the browser
```
http://localhost:3000
```
## 6. Stop the server
```bash
Ctrl + C
```
